# Artificial Intelligence Lab Programs

## Introduction

This project contains basic Artificial Intelligence programs implemented using Python.
These programs demonstrate important AI concepts such as search algorithms, human verification systems, and simple intelligent game logic.

The programs help in understanding how machines explore problem spaces and simulate intelligent behavior.

---

# List of Programs

## 1. Turing Test Simulation

This program simulates a simplified version of the Turing Test.
A bot generates responses to questions, and the program checks how machine-like the responses are.

Purpose:

* To demonstrate basic AI conversation simulation.
* To understand the concept of distinguishing between human and machine responses.

Concepts Used:

* Random response generation
* Simple decision logic
* AI simulation

---

## 2. CAPTCHA Verification System

This program generates a random CAPTCHA using letters and numbers.
The user must correctly enter the CAPTCHA to verify that they are human.

Purpose:

* To simulate a human verification system used in websites.

Concepts Used:

* Random character generation
* String manipulation
* User input validation

---

## 3. Breadth First Search (BFS)

Breadth First Search is a graph traversal algorithm that explores nodes level by level starting from the root node.

Purpose:

* To find a goal node in a graph by exploring all neighbors first.

Concepts Used:

* Graph representation
* Queue data structure
* State exploration

---

## 4. Depth First Search (DFS)

Depth First Search explores nodes by going as deep as possible along each branch before backtracking.

Purpose:

* To search for a goal node in a graph using a stack-based approach.

Concepts Used:

* Graph traversal
* Stack data structure
* Backtracking

---

## 5. Depth Limited Search (DLS)

Depth Limited Search is a variation of DFS where the search depth is limited to a fixed value.

Purpose:

* To prevent infinite exploration in deep or infinite graphs.

Concepts Used:

* Recursive search
* Depth control
* Graph traversal

---

## 6. Iterative Deepening Depth First Search (IDDFS)

Iterative Deepening Search repeatedly performs Depth Limited Search with increasing depth limits until the goal node is found.

Purpose:

* To combine the advantages of BFS and DFS.

Concepts Used:

* Iterative search
* Depth limited exploration
* Efficient memory usage

---

## 7. Tic Tac Toe Game

This program implements the classic Tic Tac Toe game using Python.
Two players take turns placing X and O on a 3×3 board.
The program checks rows, columns, and diagonals to determine the winner.

Purpose:

* To demonstrate simple game logic and decision checking.

Concepts Used:

* Game board representation
* Conditional logic
* Player turn handling

---

# Requirements

To run these programs, the following are required:

* Python 3.x
* Basic Python libraries such as:

  * random
  * string
  * collections

No external libraries are needed.

---

# How to Run the Programs

1. Install Python 3 on your computer.
2. Save each program in a separate Python file (.py).
3. Open a terminal or command prompt.
4. Run the program using the command:

python filename.py

Example:

python bfs.py
